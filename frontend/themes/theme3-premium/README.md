# Theme 3: Premium / High-End Instructor Brand

**Vibe:** Premium, confident, "top-tier driving school"

## Color Palette
- **Primary:** `#111827` (Charcoal)
- **Accent:** `#D97706` (Amber)
- **Background:** `#F9FAFB`
- **Surface:** `#FFFFFF`
- **Text Primary:** `#111827`
- **Divider:** `#E5E7EB`

## Style Guidelines
- Strong contrast, fewer colors
- Buttons: dark background + light text
- Thin dividers (1px) instead of card shadows
- Typography does most of the work
- Minimal border radius (4px)
- Clean, sharp edges

## Best For
- Marketing landing
- Instructor profile pages
- Premium lesson packages
- High-end branding

## Files
- `portal.html` - Portal login and dashboard for instructors/admin/students
- `driving_school_app.html` - Public booking calendar

## How to Use
Simply open the HTML files in your browser. All styling is self-contained within each file.
